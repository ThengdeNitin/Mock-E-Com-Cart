# Mock-E-Com-Cart

An assignment from Nexora

A full-featured **Mock-E-Com-Cart Web Application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.
It provides a seamless shopping experience with product cart management, checkout.

Features

User Features

- Add to Cart / Remove from Cart
- Checkout and Place Orders
- View Order History

System Features

- Secure REST APIs with Express.js
- MongoDB for Data Storage
- Axios for API Calls

---

Tech Stack

| Layer        | Technology                                  |
| ------------ | ------------------------------------------- |
| **Frontend** | React.js, React Router, Axios, Tailwind CSS |
| **Backend**  | Node.js, Express.js                         |
| **Database** | MongoDB with Mongoose                       |
